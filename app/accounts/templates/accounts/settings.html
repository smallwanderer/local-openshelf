Shelf-sync 빠른 시작

1
pip install shelf-sync 로 CLI 도구를 설치합니다.
2
위에서 API 토큰을 발급받고 복사합니다.
3
shelf-sync init --token YOUR_TOKEN --server {{ request.get_host }} 로 초기화합니다.
4
shelf-sync watch ~/Documents 로 폴더 동기화를 시작합니다.
